# Security Policy

## Reporting a Vulnerability
If you discover a security issue (e.g., exposed secrets, unsafe file handling, or dependency vulnerabilities), please open a private report by contacting the maintainers via GitHub.

Please do not publicly disclose sensitive details until a fix is available.

## Scope
This repository provides research code for computational modeling and analysis. It is not intended for clinical, diagnostic, or therapeutic decision-making.

## Supported Versions
Only the latest version on the main branch is supported.
